מי אנחנו

CyJimm מובילה פרויקטי אבטחה ותשתיות לארגונים שרוצים להיות ממוקדים תוצאה. אנחנו משלבים ניסיון Hands-On, אוטומציה חכמה, ומתודולוגיה של "Secure by Default".

Windows Server / AD / GPO Cloudflare / Zero-Trust Linux • Kali • Wazuh Azure / M365 / Exchange CI/CD • GitHub Actions

הצעת ערך

אנחנו לא מוכרים שקופיות. אנחנו בונים מערכות: סטנדרטיזציה, הקשחות, ניטור, ותהליכי תגובה — עם דוקומנטציה מסודרת והעברה מסודרת לצוות.

  • Baseline אבטחה לארגון תוך 14–30 יום.
  • Runbooks ו-Playbooks בעברית/אנגלית.
  • תמחור שקוף לפי תכולה, לא לפי "רשיון קסם".
